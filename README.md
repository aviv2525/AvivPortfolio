# Aviv Malul — Personal Portfolio

Personal portfolio website showcasing my projects, skills, and background as a Full Stack Developer.

**Live site:** [aviv-web.vercel.app](https://aviv-web.vercel.app) *(update with your actual URL)*

---

## Tech Stack

- **Framework:** Next.js 15
- **UI:** React 19, Tailwind CSS 4
- **Animation:** Lottie React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

```bash
npm run build   # Production build
npm run start   # Start production server
```
